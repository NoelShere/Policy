01 — OVERVIEW

Overview

Code Theory: Pocket Edition ("the App") is an offline programming reference dictionary for Android. This Privacy Policy describes how the App handles — or more accurately, does not handle — your personal information.

TL;DR: Code Theory does not collect, transmit, store, or share any personal data. The App works entirely offline on your device. No accounts, no tracking, no ads.

By installing or using Code Theory: Pocket Edition, you acknowledge and agree to this Privacy Policy. If you do not agree, please uninstall the App from your device.

100% Offline No Account Required No Data Collection No Ads No Tracking
02 — DATA WE COLLECT

Data We Collect

Code Theory: Pocket Edition collects no personal information whatsoever. The App operates completely on-device. All code examples, concept definitions, and language reference data are bundled within the App itself — nothing is fetched from a remote server.

03 — DATA WE DON'T COLLECT

Data We Don't Collect

To be absolutely transparent, here is an explicit list of data the App has no capability to collect, regardless of how it is used:

  • Your name, email address, or any contact information
  • Your device's IMEI, serial number, or hardware identifiers
  • Your IP address or network information
  • Your physical or precise GPS location
  • Your browsing history or behavior outside the App

  • Microphone or camera input
  • Contacts, calendar, or call logs
  • Payment or financial information
  • Biometric or health data
  • Advertising identifiers (Google Advertising ID or similar)
The App does not contain any advertising SDKs, crash reporting services, or analytics libraries. There is no backend server associated with Code Theory: Pocket Edition.

Code Theory: Pocket Edition does not integrate with, embed, or communicate with any third-party services. Specifically, the App does not use:

  • Google Analytics, Firebase Analytics, or any analytics platform

06 — DATA STORAGE

Data Storage & Security

All application data — including the complete programming concept database, code snippets, and language reference content — is stored entirely within the App's installation package (.apk) on your device.

No data is ever transmitted off your device. There are no servers, no databases, and no cloud infrastructure associated with this App.

The App does not create or write any files to your device's shared storage. Any operational data (such as your last selected language filter or scroll position) resides solely in Android's standard app-private storage and is automatically removed when you uninstall the App.

Because no personal data is collected or transmitted, there is no risk of a data breach affecting your personal information through Code Theory: Pocket Edition.

07 — CHILDREN'S PRIVACY

Children's Privacy

Code Theory: Pocket Edition is a general-purpose educational reference tool intended for users of all ages, including those under 13. The App is fully compliant with the Children's Online Privacy Protection Act (COPPA) and similar legislation worldwide.

Since the App collects no personal information from any user, it poses no risk to children's privacy. No age verification, parental consent, or account creation is required.

Parents and guardians can be confident that Code Theory: Pocket Edition does not collect, share, or monetize any information from their children.
